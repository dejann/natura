About

Slider images:

This template use Flex Slider to display slider images on about page. If you want to change them, simply upload your images in images/flexslider folder and link to them in index.html file. If you want to add more slides simply add more "li" tags. You can find more about Flex Slider customization on their site.

Clients

Client images

In this section you can change your clients images and link to their sites.

Team

Team images

Upload your team images into images folder and link to them in index.html file. Images with the class "top" should be grayscale. Crop your images to 380px x 254px for the best color hover effect.

Video

Video settigns

In this section you can display your Vimeo or You Tube videos. Copy your Viemo or You Tube code inside "video-wrap" div. This will make your video responsive.

Services

Your services

In this section you can display services that you offer. This section use Font Awesome to display your service icons. If you want to change them go to their website and chose ones that you like.

Quote

Quote rotation:

If you want to change quote speed rotation go to custom.js find "// quote rotator" and change the spped of rotation in miliseconds. Default is 6000ms (6s).

Portfolio

Portfolio images

In this section you can display some of your work. Go to images/portfolio/thumbs/ and upload thumbs that you want to be displayed. Natura use Quicksand to arange different types of your work. class="group1" stands for pictures class="video" for videos class="iframe" for iframed content.

    • Contact

    Update contact form with your email address

    Open send.php file and update "$adminemail" with your email address.

    Google map:

    Google map is iframed. If you want to change it, go to google maps, enter your location, click on link button and pick up the iframed version of the link. Update your google map by replacing only the src path inside index.html file. Leave id="map" inside iframe so map could be responsive.

    Colors

    Natura comes with 10 predefined colors. If you want to change template color, simply update stylesheet inside index.html file to point to your favorite color.

    Twitter

    Twitter feed use Seaofclouds with Tweet.js-Mod in order to display tweets. Since Twitter upgraded their API to 1.1 you will need to create apication to get your consumer key, consumer secret, user token and user secret. Here is a quick video of how to create your aplication. Once you get your API keys go to js/twetter/index.php file and update the file with your keys.

    Patterns

    This tamplate comes with 10 diffrerent types of patterns with 10 different opacities. Look inside images/patterns folder. First part of file name is pattern type. Second part is pattern opacity. 10 stands for 10% opacity and 100 stands for 100% (full fill) color. Bellow is the example of patterns with 50% opacity
